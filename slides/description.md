# KC-CLI Demo

This is a demo project for the KC-CLI. It is a simple project that demonstrates the basic features of the KC-CLI.

## Description

You will learn how to create new training content using the Knowledge center Command Line interface (KC-CLI).

## Prerequisites

- [Node.js](https://nodejs.org/en/)
- Basic knowledge of Markdown
- Basic knowledge of HTML

## Subjects

- Introduction to KC-CLI
- Agenda Overview
- Markdown Basics
- Using Layouts
- Creating Mermaid Diagrams
- Working with Web Components
- Using Grids
- Flexbox Layouts
- Creating Tables
- Formatting Text
- Writing Code Blocks
- Embedding HTML
- Creating Lists
- Best Practices
- Using KC Timeline
- Custom Web Components
- Other Diagrams
- Recap
- Lab Exercises
- Questions and Answers
- Closing Remarks
